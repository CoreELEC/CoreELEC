# Build container

**Clone repo**

* `cd ~/`
* `git clone https://github.com/LibreELEC/LibreELEC.tv.git LibreELEC`

**Build container**

* `cd ~/LibreELEC`
* `docker build --pull -t libreelec tools/docker/xenial`

**Build image inside container**

* `docker run -v ~/:/home/docker -h libreelec -it libreelec`
* `cd ~/LibreELEC`
* `make image`
